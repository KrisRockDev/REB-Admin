###### смена директории
```shell
cd mysite
```
###### создание миграций
```shell
python manage.py makemigrations
```
###### применение миграций
```shell
python manage.py migrate
```
###### создание суперпользователя
```shell
python manage.py createsuperuser
```
###### запуск сервера
```shell
python manage.py runserver
```
###### создание приложения
```shell
python manage.py startapp <app_name>
```
